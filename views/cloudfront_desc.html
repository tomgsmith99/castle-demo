Castle can be deployed on the edge with AWS Cloudfront and Lambda@Edge.

To test this out, click the "register" button to initiate a registration attempt with a random user.

The registration request will go to Cloudfront, and be handled by a Lambda function with Castle embedded. You'll see what the "verdict" and risk score that Castle (via Cloudfront) returns.

For more details on how this works, or to set up Castle with your own Cloudfront distribution, please see the github repo.
